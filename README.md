# Setup Instructions

1. Install homebrew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

2. Install Chezmoi

```
brew install chezmoi
```

3. Init chezmoi

```
chezmoi init https://github.com/pwoodward/dotfiles.git
```
